Introduction

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are the fundamental technologies for building web pages.

HTML provides the structure while CSS handles the styling and layout.

<!DOCTYPE html> <html lang="en"> <head> <title>My First Page</title> </head> <body> <h1>Hello, Web!</h1> </body> </html>
  • HTML defines the content structure
  • CSS controls the visual presentation
HTML Basics

HTML uses tags to define different types of content and their relationships.

Tags are enclosed in angle brackets and usually come in pairs - an opening tag and a closing tag.

HTML5 introduced many semantic elements that make the document structure more meaningful.

  • Elements are defined by tags
  • Attributes provide additional information
  • Semantic markup improves accessibility
<article> <h2>Article Title</h2> <p>Article content goes here.</p> </article>
CSS Fundamentals

CSS works by selecting HTML elements and applying styles to them.

Styles can be applied using selectors, properties, and values.

.my-class { color: blue; font-size: 16px; margin: 20px; }
Layout and Box Model

The CSS box model is fundamental to layout and spacing in web design.

Every element in web design is a rectangular box with properties for margin, border, padding, and content.

.box { margin: 10px; border: 1px solid black; padding: 20px; width: 300px; }
  • Content is the actual material inside the box
  • Padding is the space around the content
Responsive Design

Responsive design ensures websites work well on all device sizes.

Media queries allow different styles for different screen sizes.

@media screen and (max-width: 768px) { .container { width: 100%; padding: 10px; } }
  • Use flexible grids and layouts
  • Implement responsive images
